# Contributing to OpenFormosa

Thank you for helping build Taiwan-rooted open AI.

## Project scope

OpenFormosa is focused on model training, evaluation, release documentation, and reproducibility. It does not collect community datasets through GitHub, the website, or public issue templates.

## Do not upload raw data

Do not attach audio, transcripts, images, PDFs, identity documents, private messages, enterprise data, student records, medical data, financial data, or any non-consented speech to GitHub issues or pull requests.

## Useful contributions

Useful issues and pull requests include benchmark definitions, scoring scripts, model-card improvements, training recipe notes, reproducibility fixes, safety reviews, and documentation cleanup.

## Required review

Maintainers may reject issues or pull requests that include raw private data, unclear rights, privacy risks, unsafe model behavior, or claims without reproducible evidence.
